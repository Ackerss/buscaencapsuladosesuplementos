📖 Manual de Uso — NatuBrava Reposição

🚀 Como Importar Dados

⚡

Importar via API RECOMENDADO

Clique no botão roxo "Importar via API" para buscar o estoque atual do Uniplus em tempo real, sem precisar baixar planilhas. O sistema usa automaticamente os mínimos salvos da última planilha importada.

📂

Importar Planilha Manual

Arraste e solte as duas planilhas do Uniplus na área central, ou clique na área. São necessários dois arquivos: estoque_fora_dos_limites.xlsx (estoque atual + mínimos) e Produtos.xlsx (custos e preços). Ao importar, os mínimos são salvos automaticamente para uso futuro via API.

📁

Selecionar Pasta

Clique em "Selecionar Pasta" para apontar direto para a pasta de exportação do Uniplus. Carrega os dois arquivos automaticamente se a pasta contiver os arquivos necessários. (Não funciona em todos os browsers)

💡 Dica: Na primeira vez, importe a planilha manual para registrar os estoque mínimos. Depois, use sempre o botão API para atualizações rápidas do estoque atual.

🔍 Filtros e Pesquisa

🔎

Busca por texto

Digite o nome do produto, código ou categoria. A busca é instantânea e filtra em tempo real enquanto você digita.

📦

Filtro de Categoria

Selecione uma categoria/grupo para ver apenas os produtos daquele grupo (ex: CAPSULAS, ERVAS, SEMENTES).

⚠️

Filtro de Estoque

Somente Zerados mostra só produtos com saldo = 0. Baixo/Zerados mostra zerados + abaixo do mínimo.

↔

Ajustar Colunas

Ative o modo "Ajustar Colunas" e arraste as bordas dos cabeçalhos da tabela para redimensionar as colunas conforme necessário.

✕

Limpar Filtros / Zerar Progresso / Nova Importação

Limpar Filtros remove os filtros ativos. Zerar Progresso descarta todos os itens marcados como resolvidos. ⬆ Nova Importação apaga o cache e volta à tela inicial para importar dados novos.

📊 Tabela de Produtos

⇅

Ordenar colunas

Clique em qualquer cabeçalho da tabela para ordenar por aquela coluna. Clique novamente para inverter a ordem (ascendente/descendente).

🏷️

Cores de Estoque

ZERO = produto zerado, compra urgente. BAIXO = abaixo do mínimo, atenção necessária. A coluna MK% (Markup) também usa cores: verde = margem boa, amarelo = média, vermelho = baixa.

🔍

Botão Buscar Fornecedor

Clique em "Buscar" na última coluna para copiar o nome do produto e abrir automaticamente a aba da extensão NatuBrava no Chrome para buscar fornecedores com estoque disponível.

✓

Marcar como Resolvido

Clique no botão circular ○ na primeira coluna para marcar o produto como comprado/resolvido. O item sai da lista principal e vai para o painel de "Resolvidos" na parte inferior. Clique em ↩ para desfazer.

🧩 Extensão Chrome — Busca de Fornecedores

🔌 NatuBrava Buscador de Encapsulados
🧩

O que é a extensão?

É uma extensão do Chrome instalada no navegador que trabalha em conjunto com esta página. Ela permite buscar o produto em múltiplos fornecedores (Natupharma, Florien, etc.) ao mesmo tempo, verificando quem tem estoque disponível e os preços.

▶

Como usar

1. Clique em "Buscar" na linha do produto na tabela desta página.
2. O nome do produto é copiado e a aba da extensão é ativada automaticamente.
3. A extensão busca nos sites dos fornecedores e mostra quem tem o produto com preço e estoque.
4. Finalize o pedido diretamente pelo site do fornecedor.

🔄

Fluxo completo de compras

Importar via API → Ver lista de reposição → Clicar Buscar → Extensão encontra fornecedor → Efetuar compra → Marcar produto como ✓ Resolvido na tabela.

💡 Importante: A extensão deve estar instalada no Chrome e ativada. Se o botão "Buscar" não abrir a extensão, verifique se ela está habilitada em chrome://extensions.
